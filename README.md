# Kenner's Dashboard

Personal dashboard built with Vite + React.

## Run locally

```bash
cd dashboard
npm install
npm run dev
```

Open `http://localhost:3000`

## Customize

- **Schedule & deadlines** → `src/data/config.js`
- **Quick links** → same file
- **Tasks** → managed in-browser, saved to localStorage

## Build for production

```bash
npm run build    # outputs to dist/
npm run preview  # preview the build
```
